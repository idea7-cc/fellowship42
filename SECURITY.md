# Security policy

Fellowship42 handles church membership and contribution records, so security and
privacy reports should not be filed in a public issue.

Until a dedicated security mailbox is published, contact the repository owner
privately through the hosting platform's security-advisory feature. Include the
affected revision, reproduction steps, impact, and any suggested mitigation.
Do not include real member, donor, credential, or Access token data.

The project currently supports the latest revision on the default branch. No
production service or security response SLA has been announced yet.
